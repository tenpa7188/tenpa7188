# こんにちは

インフラ運用・構築をベースに、運用課題の自動化・標準化に取り組んでいます。  
GitHubでは、FortiGate運用の安全な自動化や、Terraformを用いたログ検索基盤の構築など、実務課題を意識した成果物を公開しています。  
属人化しやすい運用を、再現可能でレビューしやすい仕組みに置き換えることを重視しています。

## 注力領域

- インフラ運用の自動化
- IaCによる構成管理と再現性向上
- 運用手順の標準化
- 安全性とレビューしやすさを意識した実装
